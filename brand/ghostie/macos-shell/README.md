# Ghostie macOS Shell Prototype

Corrected shell-focused pass.

This prototype intentionally does not design the interiors of feature tabs. It focuses on:

- macOS app icon placement
- window wrapper and title bar
- side navigation
- shared status treatment
- Settings
- Drafts, Scheduled, and History as general app pages
- approval card language
- tab handoff boundaries

Open `index.html` in a browser, or serve the `brand/ghostie` folder and visit `/macos-shell/index.html`.
